# JIOS Next - 店舗発注・棚割管理システム（プロトタイプ）

## セットアップ手順（Cursor / VSCode）

### 前提条件
- **Node.js** がインストールされていること（v18以上推奨）
  - まだの場合: https://nodejs.org/ からLTS版をダウンロード・インストール

### 手順

1. **Cursorでこのフォルダを開く**
   - Cursor → File → Open Folder → `jios-app` フォルダを選択

2. **ターミナルを開く**（Cursor内で `Ctrl + `` ）

3. **依存パッケージをインストール**
   ```
   npm install
   ```

4. **開発サーバーを起動**
   ```
   npm run dev
   ```

5. **ブラウザで開く**
   - 自動的にブラウザが開きます
   - 開かない場合は http://localhost:3000 にアクセス

### 操作の流れ
1. ポータル画面 → 「通常発注」をタップ
2. 売場選択 → カテゴリを選択
3. 棚割ビュー → 商品をタップして発注数を入力
4. 棚割/一覧/重点 のタブで表示切替え
5. 右パネルで 発注/実績/メモ/商品情報 を確認

### プロジェクト構成
```
jios-app/
├── index.html          エントリーHTML
├── package.json        依存パッケージ定義
├── vite.config.js      Vite設定
└── src/
    ├── main.jsx        Reactエントリーポイント
    └── App.jsx         メインアプリケーション
```
